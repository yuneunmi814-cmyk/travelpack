# TravelPack 개인정보처리방침 (초안)

> ⚠️ 변호사/개인정보 전문가 검토 전 초안. 실제 수집 항목·수탁자와 일치하도록 출시 직전 재검증할 것(구글 플레이 Data Safety와도 일치 필요).

[회사명](이하 "회사")은 「개인정보 보호법」 등 관련 법령을 준수하며, 다음과 같이 개인정보를 처리합니다.

**시행일자: [YYYY-MM-DD]**

## 1. 수집하는 개인정보 항목 및 방법
| 구분 | 항목 | 수집 시점 |
|---|---|---|
| 필수(가입) | 이메일, 비밀번호(암호화 저장), 닉네임 | 회원가입 |
| 소셜 로그인 | 소셜 제공자 식별자, (동의 시)이메일·닉네임 | 카카오/구글 로그인 |
| 동의 이력 | 약관·개인정보·위치·마케팅 동의 여부·시각·버전 | 가입·설정 변경 |
| 서비스 이용 | 북마크, 리뷰, 여행 기록, 관심 테마 | 기능 이용 시 |
| 위치정보 | 가이드 모드 체크인 좌표 | 체크인 시(별도 동의) |
| 푸시 | 기기 푸시 토큰 | 알림 허용 시 |
| 결제(구매자) | 거래 식별자·결제수단 정보는 **PG사가 처리**, 회사는 거래 식별자·금액만 보관 | 유료 코스 구매 |
| 정산(크리에이터) | 정산 계좌, (세무)주민등록번호·사업자번호 | 유료 판매·정산 시(별도 동의·암호화) |

## 2. 수집·이용 목적
회원 식별·인증, 서비스 제공(코스·가이드 모드·마켓플레이스), 콘텐츠 추천·개인화, 결제 및 정산, 고객 문의 대응, 부정이용 방지, (동의 시)마케팅 정보 발송.

## 3. 보유 및 이용 기간 / 파기
1. 회원 탈퇴 시 지체 없이 비식별 처리 후 **30일 경과 시 완전 파기**(배치). 단, 관계 법령상 보존 의무가 있는 경우 해당 기간 보존.
2. 가이드 모드 **체크인 좌표는 수집 후 6개월 경과 시 자동 삭제(NULL)** 합니다(위치정보 최소 보관).
3. 동의 이력은 분쟁 대응을 위해 불변 보존합니다.
4. 전자상거래법상 보존: 계약·청약철회 기록 5년, 대금결제·재화공급 기록 5년, 소비자 불만·분쟁처리 기록 3년.

## 4. 제3자 제공
회사는 원칙적으로 개인정보를 외부에 제공하지 않습니다. 다만 법령에 근거가 있거나 이용자가 동의한 경우에 한합니다.

## 6. 정보주체의 권리
이용자는 언제든지 개인정보 열람·정정·삭제·처리정지를 요구할 수 있으며, 앱 내 설정 또는 개인정보 보호책임자를 통해 행사할 수 있습니다.

## 7. 위치정보의 처리
