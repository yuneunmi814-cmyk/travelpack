## 출시 전 법적 절차 체크 (사업자 본인 수행)

1. **사업자등록** (개인/법인) — 결제·통신판매업의 전제
2. **위치기반서비스사업 신고** (방송통신위원회/KISA) — 가이드 모드(GPS) 출시 선행조건
3. **통신판매업 신고** (관할 구청) — PG 가입 시 발급되는 구매안전(에스크로) 이용확인증 필요
4. **개인정보 보호책임자(CPO) 지정** + 처리방침 게시
5. **PG 가맹 계약** (포트원/토스페이먼츠 등) — 결제 키 발급
6. **크리에이터 정산 세무** — 비사업자 지급 시 사업소득/기타소득 원천징수(3.3%)·지급명세서, 사업자는 세금계산서

## 데이터 처리 현황 (처리방침 작성 근거 — 실제 코드 기준)

- 수집: 이메일·닉네임·비밀번호(bcrypt)·소셜 식별자, 동의 이력, (가이드 모드)위치좌표, 푸시 토큰, 리뷰/북마크/여행 기록, (크리에이터)정산 계좌 — *정산계좌는 별도 동의·암호화 예정*
- 보관/파기: 탈퇴 시 soft delete → **30일 후 완전 파기**(배치), 체크인 좌표 **6개월 후 NULL**, 동의 이력은 불변 보존
- 위탁/제공: PG사(결제), 클라우드(AWS), 푸시(FCM/Google), 지도(Kakao) — 처리방침에 수탁자 명시 필요
