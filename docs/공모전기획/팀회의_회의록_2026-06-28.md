## 1. 자료조사 발표 (각자 ~10분)

### 1-1. 윤은미 — 페인포인트 검증 (시장 불만 역설계)
- 타겟을 먼저 정하지 않고 **시장의 진짜 페인포인트**부터 추적. 트리플·마이리얼트립 등 유사앱의 **별점 1~3점 리뷰 약 1,000개**를 스크래핑 분석.
- 결과: **예약·환불·결제 불만이 47%로 압도적.** 반면 **"코스 큐레이션" 불만은 2%로 사실상 없음**.
- 타겟 조사(국민여행조사·소비자원·공정위·관광공사·네이버 데이터랩)에선 *"아이 동반 가볼 만한 곳"* 검색 유입이 최다 → 어제 떠올린 효도여행은 탈락, 아이 동반 가족이 유력했음.
- **딜레마** · 페인포인트(예약·결제)와 타겟(아이 동반)이 일치하지 않아 막혀 있었음 → 최용우의 "타겟 좁히기"가 돌파구가 됨.

### 1-2. 최용우 — 경쟁사·유사 서비스 (타겟 좁히기)
- 우리가 만들려던 일반 여행팩은 **이미 한국관광공사 "구석구석" 앱**으로 존재(타겟 광범위). 2030·가족·외국인까지 다 잡으면 **쓸 사람이 없다**고 판단 → 타겟을 좁혀 **두 개 보고서**로 정리(+통합본 1개, 총 3종).
  - **① 반려동물 동반 여행** — 경쟁: 반려생활(야놀자 제휴 숙소만, 코스 없음), 댕댕여지도(**강원 한정·코스 없음**). 시장성·구매력 큼.
  - **② 취약계층 무장애 여행**(임산부·영유아·장애인·고령자·다문화) — 글로벌 사례 **필더월드(장애인·휠체어 대상, 매출 약 600만 달러≈60억)**. 국내엔 종합 서비스 없음. 장애인 여행경험률 매우 낮음(미경험 86%) → "3,000원 내고 편하게"의 수요.
- **신뢰 확보** · 3인이 직접 답사·검수는 불가 → **공공데이터(인증 데이터) 기반**으로 신뢰를 세우고, 노후/오류는 **사용자 불편 제보로 수시 업데이트**, 규모 커지면 지역 서포터즈(수수료 보상) 운영.
- 핵심: **엔진은 동일** → 둘 중 하나 먼저 만들고 나중에 "껍데기"만 바꿔 확장 가능.

### 1-3. 전도준 — 수요/공급 분석 (외국인·지역축제)
- 시장을 내국인 vs 외국인으로 분해. **내국인 시장은 대기업 다운로드 상위 포화 = 레드오션**. 정부·관광공사 방향은 **지방 분산** → 우리의 교집합도 거기.
- **외국인 매출 비중**: 1위 미국·캐나다(원거리·프리미엄), 2위 일본·3위 동남아(빈도 높고 아시아권이라 교육 비용 낮음, 중국은 2025년 데이터에 부재). → **1위(미·캐)는 탈락, 일본·동남아 공략**.
- 외국인은 한국에서 **에어비앤비·우버** 등 자국 앱을 쓰는데 **한국 최적화가 안 됨 = 공략 포인트**. 외국인 전용 서비스 부재 = **블루오션**.
- 제안: **지역축제 기반 여행팩 / 바우처(티켓) 서비스**. 유럽의 **Omio·Trainline** 등이 잘 갖춘 모델을 카피. 결제가 쉬운 한국 강점을 살려 **티켓·교통·식당을 한 번에 묶는 PB형 바우처**.
- 3대 난제: ① 결제 통합 세팅 ② 외국인 유입(홍보) ③ **MVP 가설검증**(랜딩페이지 뿌려 수요 확인).

## 2. 핵심 논쟁 — A(반려동물) vs B(외국인·지역축제)

| 구분 | **A. 반려동물 동반 여행** | **B. 외국인 × 지역축제 바우처** |
|---|---|---|
| 베이스 | 정보(코스) 기반 | 결제 간편(바우처) 기반 |
| 타겟 | 내국인 반려인 | 일본·동남아 재방문 외국인 |
| 강점 | 수요 확실·API 보유·**드래프트 1일**·과제 난이도 낮음 | **블루오션**·정부 지방분산 방향 일치·하이리스크 하이리턴 |
| 약점 | 기존 앱 "필터"로 **대체 가능**(차별화 약), 반려"여행" 시장 수치 불명확·소규모, 데이터 2024년이 최신 | **사례 없음(맨땅 헤딩)**, 지자체·식당 제휴 영업 노가다, 외국인 유입 검증 안 됨 |
| 비고 | 취약계층은 롤모델(필더월드) 있으나 국내 구매력·인식 한계로 후순위 | "유명 1군"보다 **재방문객이 찾는 2군 로컬"** 큐레이션 |

- 윤은미: A는 매력·차별화가 약하다(필터 하나면 대체). 정보 과잉 시장에서 큐레이션 불만은 없으니 **반려동물처럼 수요는 있는데 공급이 없는 틈**이 낫다고 보면서도, B의 그림이 잘 안 그려진다고 비판적 검증.
- 전도준: A는 "되는 게 눈에 보여" 쉽지만 **왜 기존 업체들이 돈을 못 버나**가 걸림. B는 어렵지만 *"문제가 많을수록 기회가 많다 / 하이리스크 하이리턴"*.
- 최용우: A 두 개는 난이도가 낮아 **하다가 엎어지면 갈아끼우면 된다**(안정 선호). 부산 영도 사례로 *"현지인이 가는 곳 ≠ 관광객이 가는 곳"* — 큐레이션 시 구분 필요.

## 3. 결정사항
1. **잠정 방향 = B. 외국인(일본·동남아) 대상 "지역축제 기반 여행팩/바우처"에 무게.** 본질은 동일한 **여행팩**, 타겟·결제 레이어만 다름.
2. **A(반려동물 동반 여행)은 폐기하지 않고 병행 대안으로 보존** — 드래프트 1일이면 가능하고 엔진 동일. B가 막히면 즉시 전환(또는 별도 창업경진대회 출품 카드).
3. **신뢰 = 공공데이터(인증 데이터) + 사용자 제보 업데이트**로 일관.
4. **수익성도 평가 배점(20점)** — 관광공사×카카오T 관광택시 제휴 사례 확인. BM(교통·숙박·식당 로열티·바우처 수수료)을 반드시 설계.
5. **타겟 지역(MVP)** · 서울 근교 + 일본·동남아 외국인이 7~8월(필요 시 9월까지) 많이 가는 축제 기준으로 선정. 1군 대도시(서울·부산) 제외, **재방문객이 가는 2군 로컬**(경주·전주·안동·강릉·가평 등) 후보. *가평=친일파 후손 토지 이슈로 전도준 반대*.
6. **조직 개편 필요(미확정)** · 단순 개발 RNR(백/프/디자인)을 넘어 **영업·제휴·마케팅·재무**까지 확장해야 함 → 추후 정리.
