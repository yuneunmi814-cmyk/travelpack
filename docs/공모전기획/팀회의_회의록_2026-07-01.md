## 1. 산출물 공유
### 윤은미 (개발·배포)
- 찐맛집 세션 업데이트, **지도·대중교통 길찾기 연동**, 관광지 페이지 작동, **검색 수정**(안 되던 것 복구), 헤더 탭 3개 상시 노출, 코타 로고 임시 반영.
- **회원가입/로그인·리뷰/평점·추천 알고리즘**(위시함 기반 + 인기순 폴백) 백엔드, 관리자 페이지(미점검), **이용약관·개인정보처리방침·면책조항**(푸터), **모바일 반응형 최적화**.
- 로그인 UI는 코스 만들기 등 필요 시점에 붙이기로 보류.

### 최용우 (디버깅)
- 버그 리포트 메모장(디스코드 공유). 주요 이슈: **전반적 느림**(사진 로딩 = API 호출 한도), 코타 로고→홈 이동 불안정, **거리(내 위치) 계산 이상**, 찐맛집 날짜 표시 불필요, 상세 페이지 미통일, 카테고리(한식/양식/디저트·카페·지역별) 필요, 인기지역·거리순 정렬 정리.

### 전도준 (디자인·레퍼런스)
- 레퍼런스 분석 발표(구석구석·태국 인플루언서·UX 사이트) — 아래 3·4절.

## 2. 협업 체계 — 개발/배포 · 디버깅 · 검수
- **윤은미 개발·배포 / 최용우 버그 리포트(전체 캡처·한 번에 모아서) / 전도준 검수·디자인** 순환. 최용우가 리포트 → 윤은미 반영 → 최용우 확인.
- 상세 페이지는 새 탭보다 **모달 카드**(중요정보 상단 + 길찾기·전화 링크). 우선 **상세 페이지 통일**부터.
- **API 운영계정 신청**(관광공사)으로 호출 한도·로딩 개선. 데이터 없는 식당이 특히 느림.

## 3. 큐레이션·기능 설계 (전도준 레퍼런스)
- **위치 기반 고도화** · 모든 화면을 내 위치/추가한 위치 기준으로. 팜플릿·팝업·상단 노출에 **위치추적 링크** 삽입.
- **축제 달력 아이콘** · 날짜별 축제 수를 보고 *"축제 많은 날"로 여행일 선정* 가능하게 축제 큐레이션에 달력 추가.
- **후기 데이터 수집** · 뉴스기사·네이버 블로그(+티스토리)에서 **지역별 후기 긁어오기**. 나아가 **태국어·일본어 후기까지** 수집해 데이터·SEO 축적.
- **상세 페이지 구성** · 축제 클릭 → 메인 이미지 크게(히어로) → 주변 관광지(그 위치 기준) → 편의시설 아이콘.
- **번역** · 큰 글씨(예: 찐맛집)는 **태국인 검수** 후 번역.

## 4. 디자인 방향 — 단순·한국적
- 구석구석은 *"사용자 기반이 아닌 개발자 기반, 아름다운 쓰레기"* → **반면교사**. **최대한 단순·직관**, 불필요 기능 제거.
- **아이콘** · 서양식 면(面) 아이콘 말고 **민화풍(굵은 검정 선) 한국적 아이콘**을 전도준이 직접 제작.
- **색감** · 빨강·파랑을 포인트로만(정치색처럼 튀지 않게), 시원한 축제 느낌 팔레트. 이미지 톤 **자동 필터 통일** 가능성 검토.
- **레퍼런스 툴** · 전도준 = 클로드 디자인·구글 스티치(피그마형)·픽시, 인플루언서/UX 레퍼런스 계속 수집. (윤은미가 CSS Design Awards 링크 공유)

## 5. 소셜 vs 웹 최적화 — 웹 우선
- 윤은미: 페이스북·인스타 계정 만들어 **지역 축제 콘텐츠를 미리 쌓아두자**.
- 전도준: 소셜은 **브랜딩(로고·색감·톤·아이콘 통일) 선행**이 필요하고 발행 경험이 없음(외주형). MVP 전 소셜은 효율이 낮음.
- **결정** · **웹 "최소한" 최적화(톤·아이콘·색감·글씨체 통일)를 먼저**, 소셜은 그 후. 전도준이 **통일 디자인 요소 + 메인 화면 HTML 더미**를 만들어 공유 → 윤은미 반영.

## 6. 소셜 로그인 — 카카오+구글+라인
- 판팁·야후재팬은 소셜 로그인 미제공 → **카카오(완료)+구글+라인** 3종 연동(라인이 일본·태국에 효율적). 네이버는 개발자센터 앱 등록 필요.

## 7. 범위·목표 재확인
- 최용우·전도준: **현재 구현 수준 + 버그·완성도만 잡아도 공모전 기준은 충족**("공모전은 사실상 끝"). 그 이상은 **사업(비즈니스 모델)** 영역.
- 팀은 공모전 너머 사업까지 갈 의향. **7월 관광공사 컨설팅 대기**(날짜 미정) — 틀을 만들어두고 컨설팅 받기. 10월까지 여유 → "거북이처럼" 퀄리티 다지기.
- 제품은 **한 사람(전도준)의 실사용 니즈에서 시작**(윤은미는 축제 페인포인트가 없음) — "개발은 자기 불편에서 시작".
